# mac_code

This repository is a personal collection of miscellaneous Python scripts and utilities created for various day-to-day use cases. Each script serves a specific purpose, such as automating tasks, processing files, or solving small problems encountered in daily life.

## Contents

- **combine_pdfs.py**: Script to merge multiple PDF files into one.
- **move_images.py**: Utility to move or organize image files.
- **github-recovery-codes.txt**: Backup of GitHub recovery codes.
- Other scripts and files for quick solutions and automation.

## Usage

Feel free to explore, modify, and use any of the scripts for your own needs. Each script is self-contained and can be run independently. Refer to comments within each file for usage instructions.
